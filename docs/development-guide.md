# Development Guide

## Prerequisites

- Godot 4.x (download from godotengine.org)
- Git (for version control)

## Installation

1. Download and install Godot 4.x from the official website
2. Clone or download this project repository
3. Open Godot and import the project.godot file

## Local Development

1. Open the project in Godot Editor
2. The main scene is main.tscn
3. Press F5 or the play button to run the game
4. Edit scenes and scripts as needed

## Build Process

- Use Godot's built-in export feature for different platforms
- Go to Project > Export and select target platform
- Configure export settings as needed

## Testing

- Run the game in Godot Editor
- Check the Godot console for errors
- Test scenes individually by opening them in the editor

## Common Development Tasks

- **Adding new scenes**: Create .tscn files in scenes/ directory
- **Adding scripts**: Create .gd files in scripts/ directory
- **Adding assets**: Place images/audio in assets/ directory
- **UI development**: Use Godot's UI nodes and custom themes
