# Architecture Documentation

## Executive Summary

This is a Godot-based RPG game project using GDScript for scripting and scene-based architecture.

## Technology Stack

| Category | Technology | Version | Justification |
|----------|------------|---------|---------------|
| Game Engine | Godot | 4.x | project.godot configuration file |
| Scripting | GDScript | - | .gd files throughout project |
| UI | Godot UI Nodes | - | Scene files with UI components |

## Architecture Pattern

Scene-based game architecture with autoload singletons for global state management.
