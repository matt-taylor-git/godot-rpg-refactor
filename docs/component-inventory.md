# Component Inventory

## UI Components

### Scenes
- main.tscn: Main game scene
- character_creation.tscn: Character creation UI
- combat_scene.tscn: Combat interface
- dialogue_scene.tscn: Dialogue system
- codex_dialog.tscn: Codex/encyclopedia UI

### UI Scripts
- BaseUI.gd: Base UI class
- CharacterCreation.gd: Character creation logic
- Combat-related scripts in scripts/ directory

## Game Components

### Managers
- GameManager.gd: Global game state
- DialogueManager.gd: Dialogue system
- CodexManager.gd: Encyclopedia system

### Models
- Item.gd: Item system
- Monster.gd: Enemy definitions
- Player character classes

## Reusable Components

- UIButton.gd: Custom button component
- UIPanel.gd: Custom panel component
- UIProgressBar.gd: Progress bar component
