# Deployment Guide

## Platforms

The game can be exported to multiple platforms using Godot's export system:

- Windows (64-bit)
- Linux
- macOS
- HTML5 (web)

## Export Process

1. In Godot editor, go to Project > Export
2. Add presets for desired platforms
3. Configure export settings:
   - Window size: 800x600
   - Stretch mode: canvas_items
   - Custom user dir: pyrpg-godot
4. Export project for each platform

## Build Requirements

- Godot 4.5 with export templates
- For Windows: Windows export template
- For Linux: Linux export template
- For macOS: macOS export template

## Distribution

- **Desktop:** Distribute .exe (Windows), .x86_64 (Linux), .app (macOS)
- **Web:** Upload to web server, requires CORS setup for assets

## Performance Optimization

- Enable asset compression in export settings
- Use appropriate texture formats
- Optimize scene loading

## Testing Deployment

- Test exported builds on target platforms
- Verify save/load functionality
- Check performance on lower-end hardware
