## Critical Directories Explained

- **assets/**: Contains all game assets (images, fonts, audio, sprites)
- **scenes/**: Godot scene files defining UI and game levels
- **scripts/**: GDScript files containing game logic
- **scripts/globals/**: Autoload scripts for global state management
- **addons/**: Third-party addons, including GUT testing framework

## Entry Points

- **Main Scene:** scenes/ui/main_menu.tscn
- **Main Script:** scripts/globals/GameManager.gd (autoload)

## Integration Points

- Scenes communicate via signals
- Global managers provide shared state
- No external API integrations
