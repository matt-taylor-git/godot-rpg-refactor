# Development Guide

## Prerequisites

- Godot 4.5 (download from godotengine.org)
- Git (for version control)

## Setup

1. Clone the repository
2. Open Godot 4.5
3. Import project: Open project.godot
4. The project should load with all scenes and scripts

## Local Development

- **Run the game:** Press F5 or click Play button in Godot editor
- **Edit scenes:** Double-click .tscn files in FileSystem panel
- **Edit scripts:** Double-click .gd files
- **Test:** Use GUT addon for unit testing

## Testing

The project includes GUT (Godot Unit Test) framework in addons/gut/

To run tests:
1. In Godot editor, go to Project > Tools > GUT
2. Configure test directories
3. Run tests

## Common Tasks

- **Add new scene:** Right-click in FileSystem > New Scene
- **Add script:** Attach to node via Inspector > Script
- **Build:** Project > Export (for deployment)

## Code Style

- Use GDScript conventions
- Follow Godot naming conventions
- Use signals for communication between nodes
