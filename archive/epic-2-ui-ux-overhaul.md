# Epic 2: UI/UX Overhaul

## Description
Focuses on implementing the new art style, including the texture-based UI, "Cinzel" font, and glow effects.

## Stories

### Story 2.1: Implement Texture-Based UI Elements
As a player, I want to see a visually appealing texture-based UI so that the game has a modern yet vintage feel.

**Acceptance Criteria:**
- 9-slice frames for panels and slots
- Transparent icons for items and equipment
- Dark, elegant, and immersive visual aesthetic
- Color palette: dark teal/green, gold/bronze, vibrant cyan/teal

### Story 2.2: Integrate Cinzel Font
As a player, I want all UI text to use the Cinzel font so that it matches the medieval fantasy theme.

**Acceptance Criteria:**
- Cinzel font loaded and applied to all UI elements
- Variable weight support for flexible text rendering
- Proper font sizing and spacing

### Story 2.3: Add Glow Effects and Highlights
As a player, I want to see selective highlights and glow effects so that important elements stand out.

**Acceptance Criteria:**
- Glow effects on interactive elements
- Vibrant highlights for key information
- Performance-optimized glow rendering

### Story 2.4: Update Main Menu Scene
As a player, I want an attractive main menu so that I can start the game with good first impressions.

**Acceptance Criteria:**
- Texture-based UI elements in main menu
- Cinzel font for all text
- Proper layout and navigation

### Story 2.5: Update Combat UI
As a player, I want the combat interface to match the new art style so that it's visually consistent.

**Acceptance Criteria:**
- Combat log with styled text
- UI elements using new textures
- Consistent color scheme in combat
